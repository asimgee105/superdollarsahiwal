# Consumer Mobile App (Flutter)

This is the consumer-facing mobile application for iOS and Android, built on Clean Architecture principles and styled dynamically using color tokens and theme updates from the backend API.

## 🛠️ Stack Components

*   **SDK**: Flutter 3.x / Dart
*   **State Management**: Flutter BLoC (Business Logic Component)
*   **Local Caching**: Hive / Isar database
*   **Networking Client**: Dio
